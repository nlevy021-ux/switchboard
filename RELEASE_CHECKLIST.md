## Environment Variables Reference

### Required
- `OPENROUTER_API_KEY` - Your OpenRouter API key (server-side only)

### Optional
- `NEXT_PUBLIC_GA_MEASUREMENT_ID` - Google Analytics 4 Measurement ID (format: `G-XXXXXXXXXX`)
- `NEXT_PUBLIC_APP_URL` - Production domain URL (e.g., `https://switchboard.ai`)
